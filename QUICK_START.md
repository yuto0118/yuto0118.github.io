# 书签导航 - 快速开始指南

## 一分钟上手

1. **准备书签文件**
   - 在Chrome/Edge中：打开书签管理器 → 点击右上角"⋮" → 选择"导出书签"
   - 保存的bookmarks.html文件放在本项目目录中

2. **导入并预览**
   - 双击`quick_import.bat`文件
   - 系统会自动导入书签并打开浏览器预览

3. **完成**
   - 预览完成后关闭命令窗口即可
   - 如需更新，重复上述步骤即可

## 效果展示

- 您的书签将按照浏览器中的文件夹结构分类展示
- 每个网站都会自动获取图标
- 左侧菜单可快速跳转到不同分类

## 常见问题

**Q: 为什么有些网站图标没有显示？**  
A: 图标是从Google的favicon服务获取的，可能某些网站无法正常获取。

**Q: 导入后如何修改样式？**  
A: 样式文件位于assets/css目录，您可以根据需要修改。

**Q: 可以在手机上使用吗？**  
A: 是的，网站采用响应式设计，可以适配各种屏幕尺寸。
